# PetCare

Software de gestión veterinaria diseñado para clínicas y consultorios de mascotas.

## Descripción

PetCare es una aplicación web que permite gestionar pacientes (mascotas), citas veterinarias, historiales médicos y tratamientos de forma centralizada y eficiente.

## Stack Tecnológico

- **Frontend**: React
- **Backend**: Node.js + Express
- **Base de Datos**: MongoDB
- **Estado**: (Por definir)

## Características (Próximas)

- Gestión de pacientes (mascotas)
- Programación de citas
- Historiales médicos
- Registro de tratamientos
- Panel de administración

## Instalación

```bash
# Clona el repositorio
git clone <url-del-repositorio>

# Instala dependencias
npm install
```

## Uso

```bash
# Inicia el proyecto en modo desarrollo
npm start
```
